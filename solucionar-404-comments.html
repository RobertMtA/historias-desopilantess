Solución para Errores 404 en Comentarios

El problema

La aplicación sigue haciendo peticiones a la URL antigua (historias-desopilantes-production.up.railway.app) en lugar de la nueva (historias-desopilantes-react-production.up.railway.app).

Solución 1: Redirigir peticiones automáticamente

Este script instalará un parche en el navegador que redirigirá automáticamente las peticiones de la URL antigua a la URL nueva.

Solución 2: Limpiar caché y almacenamiento local

Esto eliminará cualquier configuración guardada que pueda estar causando el problema.

Solución 3: Servir comentarios simulados

Como solución temporal, este script mostrará comentarios simulados para que la aplicación funcione mientras se resuelve el problema.

Código de la solución

También puedes agregar manualmente este código en la consola del navegador:

// Redirigir peticiones a la API
(function() {
    const oldUrl = 'historias-desopilantes-production.up.railway.app';
    const newUrl = 'historias-desopilantes-react-production.up.railway.app';
    
    // Guardar la función fetch original
    const originalFetch = window.fetch;
    
    // Reemplazar con nuestra función que redirige URLs
    window.fetch = function(resource, options) {
        if (typeof resource === 'string' && resource.includes(oldUrl)) {
            console.log('Redirigiendo petición de URL antigua a nueva:', resource);
            resource = resource.replace(oldUrl, newUrl);
        }
        return originalFetch.call(this, resource, options);
    };
    
    console.log('✅ Parche de redirección instalado correctamente.');
})();
